# Raw Signer Sample Configuration

The following is a sample configuration for a Raw Signer worker:

```
WORKERGENID1.TYPE=PROCESSABLE
WORKERGENID1.DISABLEKEYUSAGECOUNTER=true
WORKERGENID1.IMPLEMENTATION_CLASS=com.czertainly.signserver.module.raw.RawSigner
WORKERGENID1.NAME=RawSigner
WORKERGENID1.PROCESSINTRANSACTION=true

WORKERGENID1.CRYPTOTOKEN=EntrustSAMCryptoToken

WORKERGENID1.AUTHTYPE=NONE

WORKERGENID1.ALIASSELECTOR=org.signserver.server.aliasselectors.RequestMetadataAliasSelector
WORKERGENID1.ACCEPT_ALL_USERNAMES=true

WORKERGENID1.METADATA_PROPERTY_OVERRIDE_ALLOWED=DIGESTALGORITHM,ENCRYPTIONALGORITHM

WORKERGENID1.ALLOW_BATCHSIGNING_OVERRIDE=true
WORKERGENID1.ACCEPTED_HASH_DIGEST_ALGORITHMS=SHA-256,SHA-512
```
