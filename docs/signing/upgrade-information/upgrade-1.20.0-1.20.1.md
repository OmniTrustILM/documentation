# From 1.20.0 To 1.20.1

In release version `1.20.1` the following changes were made:

## Alerts configuration for AdES Signer

Added the ability to configure alerts for the AdES Signer module. This feature allows users to set up behavior for specific events or conditions related to the signing process, enhancing monitoring and management capabilities. For more information, refer to the [AdES Signer Alerts Properties](../ades-formats/common-properties/alerts-properties.mdx).

PAdES Signer supports specific alerts configuration related to PDF documents signing, such as handling issues with document structure and visible signatures. For more information, refer to the [PAdES Signer Alerts Properties](../ades-formats/pades-signer/pades.mdx#alerts-properties).
