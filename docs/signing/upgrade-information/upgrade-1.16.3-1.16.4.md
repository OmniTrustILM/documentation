# From 1.16.3 To 1.16.4

In release version `1.16.4` the following changes were made:

## Bug Fixes and Minor Improvements

The release includes bug fixing and minor improvements to the SignServer:
- Fix basic authentication for AdES external TSA
- Update implementation of JWS Signature Activation Data using nimbus
- Fix P11NG library for RSA keys and update Entrust SAM client
