# From 1.16.2 To 1.16.3

In release version `1.16.3` the following changes were made:

## Support for the P11NG Crypto Tokens

All the AdES workers and authorization tokens have been updated to support the new P11NG Crypto Tokens.

## Bug Fixes and Minor Improvements

The release includes bug fixing and minor improvements to the SignServer:
- Fix for signing of encrypted PDF documents
