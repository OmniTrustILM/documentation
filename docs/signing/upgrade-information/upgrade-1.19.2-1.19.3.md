# From 1.19.2 To 1.19.3

In release version `1.19.3` the following changes were made:

## Enhanced algorithm support for Entrust SAM OneTime Crypto Worker

Enhanced algorithm support for the [Entrust SAM OneTime Crypto Worker](../sam-integration/entrust-sam/crypto-workers/onetime-crypto-worker.mdx) by adding RSA 3072 support, fixing the CSR signature algorithm name for EC keys, and improving error escalation from the crypto token to make configuration issues more visible.
