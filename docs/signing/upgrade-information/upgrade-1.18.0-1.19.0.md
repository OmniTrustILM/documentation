# From 1.18.0 To 1.19.0

In release version `1.19.0` the following changes were made:

## Improved PAdES visual signature

The PAdES visual signature has been improved to support additional configuration properties for better customization of the visual appearance. For more information, refer to the [PAdES Signer Properties](../ades-formats/pades-signer/pades.mdx).

## Bug Fixes and Minor Improvements

The release includes bug fixing and minor improvements:
- Fix configuration of OCSP/CRL source for validation data
