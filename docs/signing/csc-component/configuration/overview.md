# Configuration of the CSC component

Configuration of the CSC component can be split into few parts:
- [Application configuration](./application-configuration.md) - configuration of the CSC component itself
- [Workers configuration](./workers-configuration.md) - configuration of the workers that will be used to sign the requests
- [Profiles configuration](./profiles-configuration.md) - configuration of the profiles that will be used to sign the requests

Both workers and profiles configuration are referred from the application configuration.
