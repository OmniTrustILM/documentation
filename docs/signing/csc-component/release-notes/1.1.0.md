# Release Notes 1.1.0

In release version `1.1.0` the following changes were made:

## Enhancements 🛠
* Delete one-time key asynchronously after signing operation
* Improve performance of number of replenished keys
* Improve one-time key management
* Update process of updating docker container nightly

## Bug Fixes 🔩
* Fix replenish trigger and add default cron values
